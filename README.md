<span><div><div><div><div><span style="font-size: 16pt;">零、名詞解釋</span></div><div><span style="font-size: 16pt;">貨幣流動性(money liquidity)</span></div><ol><li><div><span style="font-size: 12pt;">對央行而言，所指為基礎貨幣，其對基礎貨幣量進行發行及管理，流動性過剩代表發行了過多的基礎貨幣超過實體經濟承載量</span></div></li><li><div><span style="font-size: 12pt;">對商業銀行而言，所指為存款大於貸款的比例，若存款遠高於貸款代表流動性過剩</span></div></li><li><div><span 財政政策</span></div><div><span style="font-size: 12pt;">1.信貸政策</span></div><div><span style="font-size: 12pt;">內容主要可分四項</span></div><ul><li><div><span style="font-size: &nbsp;6.選擇性信用管制：對特定對象進行專項管理，譬如消費信用、證券信用...</span></div><div><span style="font-size: 12pt;">&nbsp;&nbsp; &nbsp;7.直接信用管制：央行直接對商行的商業活動進行直接干預以控制其信貸活動</span></div><div><br></div><div><span
